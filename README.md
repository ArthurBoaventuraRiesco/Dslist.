# Dslist - Catálogo de Jogos com Spring Boot

## Visão Geral
O **Dslist** é uma aplicação backend desenvolvida em **Java 21** com **Spring Boot**, voltada à gestão de listas de jogos.  
Ela permite consultar, cadastrar e listar jogos de forma ordenada, com integração a banco de dados e testes automatizados via **Maven** e **GitHub Actions**.

## Tecnologias Utilizadas
- **Java 21**
- **Spring Boot 3**
- **Spring Data JPA**
- **H2 Database** (para testes)
- **PostgreSQL** (para produção, opcional)
- **Maven**
- **JUnit 5**
- **MockMvc**
- **GitHub Actions (CI/CD)**
